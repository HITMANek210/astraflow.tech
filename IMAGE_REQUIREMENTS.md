# Image Requirements

## Required Images

The following images need to be added to complete the portfolio:

### 1. Professional Headshot
- **Location:** `public/images/headshot.jpg`
- **Specifications:**
  - Format: JPG or PNG
  - Recommended size: 400x400px (square)
  - Professional portrait photo
  - Neutral background, warm lighting
  - Business casual attire
- **Used in:**
  - About Intro section (rectangular display)
  - Credentials section (circular display)

### 2. Favicon
- **Location:** `public/favicon.jpg` (or `.ico`, `.png`)
- **Specifications:**
  - Format: JPG, PNG, or ICO
  - Recommended size: 32x32px or 64x64px
  - Abstract purple gradient "A" lettermark icon
  - Dark background, minimal design
- **Used in:** Browser tab icon

## Quick Start

1. Add your professional headshot to `public/images/headshot.jpg`
2. Add your favicon to `public/favicon.jpg` (or update the reference in `app/layout.tsx` if using a different format)

The site will work without these images, but they should be added for the best user experience.
